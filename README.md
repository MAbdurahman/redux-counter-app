# redux-counter-app

<p>redux-counter-app is a React.js application with the implementation of a counter with Redux</p>

### To install dependencies
```shell
npm install
```
or
```shell
yarn or yarn install
```
### To run the app
```shell
npm run start
```
or

```shell
yarn start
```

### Build the app
```shell
npm run build
```
or

```shell
yarn build
```
